# Masters Draft 2026

Live leaderboard for a 6-person Masters 2026 golf sweepstake.

Each player drafts 4 golfers before the tournament. Scores are pulled live from ESPN and combined to rank all 6 sweepers. The Sunday captain mechanic doubles one golfer's R4 score.

**Live at:** [masters-draft-26](https://carrot-consulting.github.io/masters-draft-26/)

## How It Works

- Leaderboard auto-refreshes every 10 minutes
- Click any sweeper to see their 4 golfers and individual scores
- Each sweeper names a captain before Sunday — that golfer's R4 score is doubled
